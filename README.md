# 🎯 Case Study: Multi-Phase Paid Search Architecture & Infrastructure Audit

## Project Overview
This repository contains a comprehensive Google Ads campaign build and digital infrastructure audit designed for **Angel Plumbers**, a local service business operating within high-density, high-income suburban clusters in Johannesburg. 

The primary challenge addressed by this campaign strategy is optimization against digital infrastructure constraints—specifically, maximizing lead generation traffic into a single-page website architecture without wasting ad spend.

## Core Strategic Frameworks

### 1. Phase 1: The Anchor Jump Link Strategy
Because the current website operates strictly on a single scrolling page layout, standard multi-page destination targeting was impossible. To bypass this without requiring an immediate, expensive web design rebuild, we implemented custom URL anchor paths (`/#Services`, `/#Contact`) mapped to manually deployed **Sitelink Extensions**. This structurally organizes the traffic at the search result level and forces the user's browser to instantly scroll to relevant content blocks, lowering immediate bounce rates.

### 2. Campaign Insulation & Negative Matching
To protect the daily budget from broad, low-intent search queries, a robust negative keyword matrix was deployed alongside localized phrase and exact match variations. Cross-ad group negativing ensures that high-intent emergency traffic is isolated cleanly into dedicated call-ready structures rather than bleeding into general domestic plumbing queries.

### 3. High-Value Suburb Targeting
Budget allocation is strictly prioritized toward specific premium economic hubs within Johannesburg (e.g., Sandton, Bryanston, Fourways, Randburg) using tight radius clustering rather than broad provincial targeting.

## Phase 2 Optimization Roadmap
* **Standalone Landing Page Deployment:** Reinvesting initial campaign revenue into high-converting, single-intent landing pages for top-tier service lines (Geyser Replacement & Leak Detection) to drive up Google Quality Scores and reduce average Cost-Per-Click (CPC).
* **Advanced Tracking Layers:** Configuring Google Tag Manager (GTM) triggers for explicit data capturing including mobile click-to-calls and contact form submissions to feed automated Smart Bidding algorithms.
